# Invest Tool V2 — A股全历史走势分析工具

> 输入任意A股股票代码，自动生成上市以来全历史走势图、大事记标注、行业信息与年度统计。

## 功能

- **交互式走势图**：Plotly 交互图表，支持缩放、平移、悬停查看详情
- **5 层子图**：股价+均线 | 成交量 | 每日涨跌幅 | 年度涨跌幅 | 年度价格区间
- **自动大事记**：上市首日、历史最高价、暴涨暴跌日、重大市场事件
- **公司信息**：行业板块、董事长、公司简介（来自东方财富F10）
- **名称搜索**：支持输入股票名称或代码自动联想
- **一键启动**：双击 `start_tool.bat` 即可运行

## 快速开始

```bash
# 安装依赖
pip install flask requests pandas numpy plotly

# 一键启动
start start_tool.bat

# 或手动启动
python stock_history_server.py
```

打开浏览器访问 **http://localhost:6789**

## 数据来源

| 数据 | 来源 |
|------|------|
| K线行情 | 腾讯证券 API |
| 公司信息/行业 | 东方财富 F10 |
| 搜索建议 | 东方财富搜索 |

## 技术架构

```
用户浏览器 → Flask Web服务器 → 腾讯证券API (K线) + 东方财富F10 (公司信息)
                ↓
          Plotly 生成交互式 HTML 图表
                ↓
          iframe 嵌入展示
```
